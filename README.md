## 本リポジトリのビルド方法

原稿はrst形式で書かれており、sphinxを経由してPDFを生成することがきます。sphinx 1.3.3+日本語化パッチと、dvipdfmxがインストールされた環境で、 `` make latexpdfja`` を実行するとPDFが出力されるはずです。
